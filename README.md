
  # Figma to Code Demo

  This is a code bundle for Atlantis Web Canvas. The original project is available at https://www.figma.com/design/BQJ42sC27ckwMGH1u4kyln/Atlantis-Web-Canvas. It gives you an empty UI Shell to practice moving from Figma into Code.

  ## Running the project

  1. Open the project in Cursor
  2. Press Cmd + J on your keyboard to open Terminal
  4. Run `npm i` to install the dependencies.
  5. Run `npm run dev` to start the development server.
  6. Cursor's browser should open and you'll see the UI Shell
  7. Prompt away!
